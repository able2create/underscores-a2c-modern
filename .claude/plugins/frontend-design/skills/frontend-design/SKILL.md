## Frontend Design Skill

Create distinctive, production-grade frontend interfaces that avoid generic AI aesthetics.

### Before You Code

Understand the context:

- **Purpose**: What problem does this solve? Who is the target audience?
- **Tone**: What aesthetic direction fits? (minimalist, maximalist, retro-futuristic, brutalist, luxury, playful, art deco, etc.)
- **Constraints**: What frameworks, libraries, or accessibility requirements apply?
- **Differentiation**: What one thing will make this design unforgettable?

Commit to a BOLD aesthetic direction. Don't default to conventional choices.

### Typography

Choose fonts that are beautiful, unique, and interesting. Avoid generic fonts like Arial and Inter; opt instead for distinctive choices that elevate the frontend's aesthetics.

### Color & Theme

Build cohesive color palettes using CSS variables. Dominant colors with sharp accents outperform timid, safe combinations.

### Motion & Animation

Implement through CSS-first solutions. Prioritize high-impact page load sequences, scroll-triggered reveals, and meaningful transitions over scattered micro-interactions.

### Spatial Composition

Use unexpected layouts with asymmetry, overlap, diagonal flows, and grid-breaking elements. Avoid predictable, safe grid structures.

### Visual Details

Create atmosphere through contextual effects, textures, gradients, decorative patterns, and depth that match the aesthetic vision.

### What to Avoid

Never produce:

- Generic font families (Arial, Helvetica, Inter without purpose)
- Clichéd color schemes (especially purple gradients)
- Predictable, cookie-cutter layouts
- Generic component patterns that lack context-specific character

### Implementation

Match code complexity to the aesthetic vision:

- Maximalist designs warrant elaborate animations and visual layers
- Minimalist designs demand precision in spacing, typography, and restraint

Claude is capable of extraordinary creative work. Don't hold back from bold, context-specific design decisions.
